# Drum Kit

An interactive Drum Kit web application built using HTML, CSS, and JavaScript.

## Overview

This project allows users to play different drum sounds either by clicking the on-screen buttons or by pressing corresponding keyboard keys. It demonstrates JavaScript event handling, DOM manipulation, and audio integration.

## Features

- Play drum sounds using mouse clicks
- Keyboard support for faster interaction
- Button animation effects
- Responsive and interactive user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Concepts Practiced

- Event Listeners
- DOM Manipulation
- Audio Objects
- Keyboard Events
- CSS Animations

## Learning Outcome

Through this project, I gained hands-on experience in handling user interactions, working with audio files, and creating dynamic web applications using JavaScript.
